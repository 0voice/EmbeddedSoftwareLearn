# 2025 嵌入式软件开发全景笔记 📘

## 概述
欢迎来到本项目，你将收获一份系统、全面且贴近实战的嵌入式软件开发学习路线和知识点总结。  
涵盖范围包括 C/C++、驱动开发、RTOS、嵌入式 Linux、网络通信与物联网、常用工具链等。
